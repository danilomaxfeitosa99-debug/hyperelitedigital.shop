# M & M Refeições Coletivas - Site Institucional

Este é um site institucional estático de alta performance desenvolvido para a empresa **M & M Refeições Coletivas LTDA** (CNPJ: 65.197.852/0001-66). O projeto foi criado com foco em alimentação industrial e nutrição corporativa, com total responsividade.

## 🚀 Estrutura do Projeto

```text
mm-refeicoes-65197852000166/
├── index.html      # Página principal (Home, Sobre, Serviços, Contato)
├── css/
│   └── style.css   # Estilização focada em saúde e profissionalismo
├── js/
│   └── script.js   # Funcionalidades interativas (Scroll, Menu Mobile, Formulário)
└── README.md       # Documentação do projeto
```

## ✨ Funcionalidades

- **Design Premium:** Estética focada em saúde e nutrição, com cores verdes e tipografia moderna.
- **Totalmente Responsivo:** Otimizado para Mobile, Tablet e Desktop.
- **Scroll Suave:** Navegação fluida entre as seções da página.
- **Header Dinâmico:** Muda de aparência ao rolar a página para melhor legibilidade.
- **Menu Mobile:** Navegação intuitiva em dispositivos móveis.
- **Simulação de Formulário:** Validação básica e feedback visual de envio de proposta corporativa.

## 🛠️ Como Usar

1. **Edição de Conteúdo:** Abra o arquivo `index.html` em qualquer editor de texto para alterar textos, imagens e informações de contato.
2. **Personalização de Cores:** As cores principais podem ser alteradas no arquivo `css/style.css` através das variáveis `:root`.
3. **Deploy:** Basta fazer o upload de todos os arquivos para qualquer servidor de hospedagem.

## 📝 Notas de Desenvolvimento

- Desenvolvido em **HTML5**, **CSS3** e **JavaScript Puro**.
- Sem dependências de frameworks externos.
- Imagens otimizadas via Unsplash para carregamento rápido.
- Links externos configurados para redirecionamento padrão (`https://google.com`).
